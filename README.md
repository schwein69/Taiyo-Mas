# Taiyo-Mas
TAIYO-MAS is an intelligent system for photovoltaic energy management. The project simulates a residential microgrid composed of photovoltaic generation, battery storage, household consumption, and grid interaction. Through configurable operating modes (Balanced, Direct, and Selling), autonomous agents coordinate energy distribution and optimization strategies. Built in Kotlin, the system combines discrete-time simulation, BDI agent-based decision making (Jason/AgentSpeak), and automated planning techniques(Strips) to explore adaptive and efficient energy management in smart microgrids.
